# 원문 자료 폴더 (public/originals)

각 사업의 **원문 RFP 파일(원본 그대로)** 을 이 폴더에 넣으면, 해당 사업의 「사업내용」
화면에서 사업명 옆 **[원문 다운로드]** 버튼으로 원본 파일명 그대로 내려받습니다.

## 새 원문을 추가/교체할 때
1. **원본 파일명 그대로** 이 폴더(`ocean-rfp-web/public/originals/`)에 복사 (이름 바꾸지 말 것)
2. `src/data/projects.js`의 해당 사업 `download.file` 값을 **정확한 파일명**으로 적고 `has: true`
3. `bun run build` (dev 서버는 자동 반영)

## 동작 원리
- `public/` 폴더 파일은 빌드 시 사이트 루트로 복사됩니다.
- 버튼은 `<a href="originals/<URL인코딩된 파일명>" download="<원본 파일명>">` 으로 연결되어
  공백·괄호·한글이 들어간 원본 파일명도 그대로 다운로드됩니다.
