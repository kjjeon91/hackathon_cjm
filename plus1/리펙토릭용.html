    결과보고서 생성 설정

    결과보고서 본문은 이 화면에 표시하지 않고 별도 HTML/DOCX/HWP/PDF 파일로 생성합니다. 현재 프로토타입에서는 샘플 결과보고서 HTML을 별도 파일로 제공합니다.
    샘플 결과보고서 HTML 열기

    ※ 프로토타입에서는 샘플 결과보고서를 별도 HTML 파일로 제공합니다. 실제 구현 시 DOCX/HWP/PDF 템플릿으로 자동 출력하고 그룹웨어 결재 API와 연동합니다.

    결재 상신 흐름

    1차최흥배 부서장 확정
    2차오주혜 선임 확인
    3차송용식 부사장 검토
    최종대표 결재
