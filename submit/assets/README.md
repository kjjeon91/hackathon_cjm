# 재사용 자산 (assets) — 남이 그대로 가져다 쓸 수 있게

> 재현·전파성(25점)의 핵심. **이 폴더만 보고 다른 팀이 따라 할 수 있어야** 높은 점수.

여기에 넣을 것 (해당되는 것만):
- 핵심 프롬프트 모음 (`prompts.md`)
- 사용한 스킬 / 서브에이전트 구성
- 프로젝트 `CLAUDE.md` 사본 (어떻게 세팅했는지)
- MCP·외부도구 연동 설정·사용법 메모
- "처음 보는 사람이 10분 안에 실행하는 법" 한 페이지

> 팁: 거창할 필요 없음. **실제로 쓴 프롬프트와 셋업을 그대로** 남기는 게 가장 좋은 자산입니다.
